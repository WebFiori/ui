# XML Output

This example demonstrates generating XML documents from HTMLNode trees.

## What This Example Shows

- Rendering nodes as XML with `toXML()` and `HtmlRenderer::renderXML()`
- XML declaration and proper attribute quoting
- Self-closing void elements in XML mode
- Building structured XML data

## Files

- [`example.php`](example.php) - Main example code

## Running the Example

```bash
php example.php
```

## Expected Output

Well-formed XML documents with proper declarations, quoted attributes, and self-closing tags.
