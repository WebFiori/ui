# Elements and Attributes

This example demonstrates how to create HTML elements, set attributes, chain methods, and iterate over children.

## What This Example Shows

- Creating elements with constructor attributes
- Method chaining for fluent API
- Setting styles as arrays
- Boolean (valueless) attributes
- Iterating children with `foreach`
- Nesting elements

## Files

- [`example.php`](example.php) - Main example code

## Running the Example

```bash
php example.php
```

## Expected Output

Several HTML elements demonstrating attributes, styles, nesting, and child iteration.
