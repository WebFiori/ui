# WebFiori UI Examples

This directory contains practical examples demonstrating how to use the WebFiori UI library.

## Examples Overview

| # | Example | Description |
|---|---------|-------------|
| 01 | [basic-document](01-basic-document/) | Creating a complete HTML document |
| 02 | [elements-and-attributes](02-elements-and-attributes/) | Building elements, chaining, iteration |
| 03 | [forms-and-input](03-forms-and-input/) | Form creation with various input types |
| 04 | [tables](04-tables/) | Table creation with rows, columns, and data |
| 05 | [html-templates](05-html-templates/) | Loading HTML templates with slot variables |
| 06 | [php-templates](06-php-templates/) | Loading PHP templates with passed variables |
| 07 | [standalone-renderer](07-standalone-renderer/) | Using HtmlRenderer for per-instance config |
| 08 | [xml-output](08-xml-output/) | Generating XML documents from nodes |

## Prerequisites

- PHP 8.1 or higher
- Composer dependencies installed

## Running Examples

Each example is self-contained and can be run independently:

```bash
cd examples/01-basic-document
php example.php
```

Or from the project root:

```bash
php examples/01-basic-document/example.php
```
