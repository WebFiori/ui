# Tables

This example demonstrates how to create and manipulate HTML tables.

## What This Example Shows

- Creating tables with specified rows and columns
- Setting cell content
- Adding rows and columns dynamically
- Styling tables

## Files

- [`example.php`](example.php) - Main example code

## Running the Example

```bash
php example.php
```

## Expected Output

An HTML table with headers and data rows.
