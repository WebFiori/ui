# Forms and Input

This example demonstrates how to create HTML forms with various input types.

## What This Example Shows

- Creating form elements with action and method
- Adding labeled input fields
- Different input types (text, password, email, select, textarea)
- Required and placeholder attributes

## Files

- [`example.php`](example.php) - Main example code

## Running the Example

```bash
php example.php
```

## Expected Output

A complete login form and a contact form with various input types.
